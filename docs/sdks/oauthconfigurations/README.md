## getConnectorType

Get details for a specific OAuth-capable connector type.

### Example Usage

<!-- UsageSnippet language="typescript" operationID="getOAuthConnectorType" method="get" path="/oauth/registry/{connectorType}" -->
```typescript
import { Pipeshub } from "pipeshub";

const pipeshub = new Pipeshub({
  serverURL: "https://api.example.com",
  bearerAuth: process.env["PIPESHUB_BEARER_AUTH"] ?? "",
});

async function run() {
  const result = await pipeshub.oauthConfigurations.getConnectorType({
    connectorType: "<value>",
  });

  console.log(result);
}

run();
```

## deleteConfiguration

Delete an OAuth configuration.<br><br>
<b>Warning:</b><br>
Cannot delete if the configuration is used by active connectors.
Disable or delete dependent connectors first.


### Example Usage

<!-- UsageSnippet language="typescript" operationID="deleteOAuthConfig" method="delete" path="/oauth/{connectorType}/{configId}" -->
```typescript
import { Pipeshub } from "pipeshub";

const pipeshub = new Pipeshub({
  serverURL: "https://api.example.com",
  bearerAuth: process.env["PIPESHUB_BEARER_AUTH"] ?? "",
});

  const result = await pipeshub.oauthConfigurations.deleteConfiguration({
    connectorType: "<value>",
    configId: "<id>",
```
