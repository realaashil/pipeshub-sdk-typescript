# AuthConfig

## Overview

### Available Operations

* [getMicrosoft](#getmicrosoft) - Get Microsoft authentication configuration

## getMicrosoft

Get Microsoft authentication configuration.

### Example Usage

<!-- UsageSnippet language="typescript" operationID="getMicrosoftAuthConfig" method="get" path="/configurationManager/authConfig/microsoft" -->
```typescript
import { Pipeshub } from "pipeshub";

const pipeshub = new Pipeshub({
  serverURL: "https://api.example.com",
  bearerAuth: process.env["PIPESHUB_BEARER_AUTH"] ?? "",
});

async function run() {
  const result = await pipeshub.authConfig.getMicrosoft();

  console.log(result);
}

run();
```

### Standalone function

The standalone function version of this method:

```typescript
import { PipeshubCore } from "pipeshub/core.js";
import { authConfigGetMicrosoft } from "pipeshub/funcs/auth-config-get-microsoft.js";

// Use `PipeshubCore` for best tree-shaking performance.
// You can create one instance of it to use across an application.
const pipeshub = new PipeshubCore({
  serverURL: "https://api.example.com",
  bearerAuth: process.env["PIPESHUB_BEARER_AUTH"] ?? "",
});

async function run() {
  const res = await authConfigGetMicrosoft(pipeshub);
  if (res.ok) {
    const { value: result } = res;
    console.log(result);
  } else {
    console.log("authConfigGetMicrosoft failed:", res.error);
  }
}

run();
```
