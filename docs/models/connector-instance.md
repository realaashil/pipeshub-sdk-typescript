# ConnectorInstance

A configured connector instance. Represents an active or configured
connection to an external service.


## Example Usage

```typescript
import { ConnectorInstance } from "@pipeshub/sdk/models";

let value: ConnectorInstance = {
  connectorId: "conn_abc123",
  connectorType: "google-drive",
  instanceName: "Company Google Drive",
  scope: "team",
};
```
