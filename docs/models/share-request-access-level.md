# ShareRequestAccessLevel

Permission level for shared users:
<ul>
<li><code>read</code> - Can view only</li>
<li><code>write</code> - Can add messages</li>
</ul>


## Example Usage

```typescript
import { ShareRequestAccessLevel } from "@pipeshub/sdk/models";

let value: ShareRequestAccessLevel = "read";
```
