# CustomScheduleConfig

Run crawling job using a custom cron expression for complex schedules

## Example Usage

```typescript
import { CustomScheduleConfig } from "@pipeshub/sdk/models";

let value: CustomScheduleConfig = {
  scheduleType: "custom",
  cronExpression: "0 2 * * *",
  description: "Run daily at 2:00 AM",
};
```
